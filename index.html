Intro

Full-stack web developer utilizing a bachelor’s degree in psychology to create immersive UX/UI design. Currently earning a full-stack developer certificate through UNC-Chapel Hill’s coding bootcamp. Strengths include HTML, CSS, JavaScript, responsive web design, and back-end integration. Known for my leadership qualities and ability to take initiative. Successfully led a large organization while in university, restructuring its design while also increasing its membership and community outreach. Recently worked in two teams, one as a back-end developer creating an LiveLink utilizing API integration to deliver the user new and exciting date-night meal recipes, and one as a front-end developer to create a coffee shop LiveLink that utilized many different technologies, such as MySQL and Handlebars, to allow users to order and receive coffee in a COVID-friendly manner. My adaptive nature and positive interpersonal communication skills make me a great addition to any development team!

Dinner Dice

LiveLink Code

An application where users may select specific dietary options and obtain a random food and drink combination based off their selection. Day Planner

LiveLink Code

Coding Quiz

LiveLink Code

A web-based application that allows the user to take a simple quiz to test their coding knowledge. Scores are saved to local storage. Technologies used: HTML, CSS, JavaScript, and JSON.
